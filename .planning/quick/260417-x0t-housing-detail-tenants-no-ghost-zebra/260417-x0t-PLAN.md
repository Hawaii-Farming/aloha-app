# Quick Task 260417-x0t

## Problem

Tenants grid in Housing detail shows striped bands to the right of the last column AND below the last row. Stripes come from the intentional "ghost zebra fill" in `app/styles/kit.css` (`[data-ag-theme-mode='light'].ag-grid-cell-borders .ag-body-viewport` → `repeating-linear-gradient`) painted on the AG Grid body viewport. The pattern is desirable for long list grids that need to read as a table to the edge; on a short detail sub-grid it reads as dead striped space.

## Fix

File: `app/components/crud/housing-detail-view.tsx` (TenantsGrid)

- `domLayout='autoHeight'` — grid height matches content, so no empty striped area below the last row.
- `onGridReady` → `sizeColumnsToFit()` — columns stretch to fill grid width, so no empty striped area to the right of the last column.
- Add `GridReadyEvent` to the ag-grid-community type import.

## Verification

`pnpm typecheck` clean. Tenants grid renders with stripes only underneath actual rows.
