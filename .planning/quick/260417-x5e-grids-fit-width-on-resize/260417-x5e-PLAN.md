# Quick Task 260417-x5e

## Objective

All AG Grid list views should fit their columns to the available width, and re-fit automatically when the viewport width changes (sidebar collapse/expand, window resize).

## Change

File: `app/components/ag-grid/ag-grid-wrapper.tsx`
