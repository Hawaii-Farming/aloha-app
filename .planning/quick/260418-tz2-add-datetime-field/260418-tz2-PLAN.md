# Quick Task 260418-tz2

## Objective

Add `type: 'datetime'` form field so timestamp columns (scheduler, task tracker) can capture hh:mm alongside the date. Keep pure `'date'` type for DOB, check dates, etc.

## Changes

### `app/lib/crud/types.ts`

- Add `'datetime'` to `FormFieldType` union.

### `packages/ui/src/kit/form-fields.tsx`

- Add `FormDateTimeField` — calendar popover + `<Input type="time">` inline.
- Storage format: `formatISO(date)` — ISO 8601 with local offset (e.g. `2026-04-16T09:00:00-10:00`), round-trips via `parseISO` in any TZ.
- Export alongside existing `FormDateField`.

### `app/lib/crud/render-form-field.tsx`

- Add `case 'datetime':` → render `FormDateTimeField`.

### `app/lib/crud/ops-task-schedule.config.ts`

- `start_time` and `stop_time`: `type: 'date'` → `type: 'datetime'`.

### `app/lib/crud/ops-task-tracker.config.ts`

- `start_time` and `stop_time`: `type: 'date'` → `type: 'datetime'`.

## Verification

- `pnpm typecheck` clean.
- Manually: scheduler create — pick a date, set time 09:00, submit, reopen — same date + time displayed.

## Out of scope

- 7-day batch scheduler form (separate design task).
- Column display formatting for datetime columns in list views.
