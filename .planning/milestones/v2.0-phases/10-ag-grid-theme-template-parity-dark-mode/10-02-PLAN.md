    ```tsx
      <div className="hidden md:block"><WorkspaceSidebar ... /></div>
    ```

    to:

    ```tsx
    <div className="flex min-h-0 flex-1 overflow-hidden">
      <div className="hidden md:block"><WorkspaceSidebar ... /></div>
    ```

    Key changes:
    1. Add `min-h-0` to the outer flex row (prevents it inheriting content height from grandparent).

    4. Inner `<div>` gets `min-h-0` so its `flex-1` resolves correctly.
