**button.tsx — Add pill variant (per D-05):**

Add a new `pill` entry to the `variant` object in `buttonVariants` CVA, after the existing `link` variant:

```typescript
pill: 'rounded-full bg-primary text-primary-foreground border border-primary-foreground/20 px-8 py-2 font-medium hover:bg-primary/90',
```

This gives:
- `rounded-full` = 9999px border-radius (pill shape)
- `bg-primary text-primary-foreground` = near-black bg with near-white text in dark, inverted in light
- `border border-primary-foreground/20` = subtle border at 20% opacity of the foreground color
- `px-8 py-2` = generous horizontal padding matching DESIGN.md "8px 32px"
- `font-medium` = weight 500
- `hover:bg-primary/90` = subtle hover darken

The pill variant is a NEW variant — it does not modify existing variants. The `size` variants (default, sm, lg, icon) still apply to pill buttons.
